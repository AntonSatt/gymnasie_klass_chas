# Slides (gymnasie)

Det här är innehållet som publiceras till GitHub Pages: en landningssida + interaktiva Reveal.js-presentationer för gymnasieelever.

- Live: `https://antonsatt.github.io/gymnasie_klass_chas/`

## Innehåll

- `content/dopamine-trap.md` – markdown-slides
- `content/ai-kontext/` – standalone HTML/CSS/JS med live-demo (OpenRouter)
- `data/slides.json` – metadata för markdown-presentationer

## Tech stack

- Reveal.js
- HTML/CSS/JavaScript
- Tailwind (landningssidan)
- Webpack-build för markdown-slides
- OpenRouter API (live-demo)
