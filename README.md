# Gymnasie-presentationer (Chas Academy)

Det här repot innehåller interaktiva Reveal.js-presentationer för gymnasieelever, plus en enkel landningssida som listar dem.

- Live (GitHub Pages): `https://antonsatt.github.io/gymnasie_klass_chas/`

## Vad finns här?

- **Dopaminfällan: AI & din hjärna** – markdown-baserade slides
- **AI & Kontext: Nyckeln till Smarta Svar** – fristående HTML/CSS/JS med live-demo via OpenRouter

## Tech stack

- Reveal.js
- HTML/CSS/JavaScript
- Node.js toolchain (webpack) för markdown-slides/bygg
- Tailwind (landningssidan)
- OpenRouter API (för live-demo i “AI & Kontext”)

## Struktur (kort)

- `slides-content/` – webbsidan + presentationskoden
- `slides-content/content/` – själva presentationerna
